Features Overview

1. Login & Logout

Description: Implement a login and logout feature connected to MySQL.

  • Login Form: A simple form with username and password fields.
  • Logout Button: A button to log out the user.

2. View Data as Table and Card

Description: Display data in both table and card formats.

  • Table View: A responsive table layout.
  • Card View: A grid layout with cards displaying data.

3. Search By Id

Description: Search for data by ID.

  • Search: Search data by Id.
  • Display Results: Show data corresponding to the ID.

4. CRUD Operations

Description: Implement create, read, update, and delete operations.

  • Create: Form to input and submit new data.
  • Read: Display existing data in table or card view.
  • Update: Form pre-filled with data for editing.
  • Delete: Button to delete data, with confirmation prompt.

5. Download

Description: Download Complete Data As excel File.

  • Download: Download the data in excel format.
